VIDEOBOOK PROJECT

Applicazione sviluppata in C# (NetworkFramework 4.5) per la gestione di libreria, videoteca, software e musica
Sfrutta un DB MYSQL il cui file di creazione e lo schema sono inclusi nel progetto.

Elenco delle Librerie da Scaricare e da Importare per il corretto funzionamento:

- Mysql Net Connector : http://dev.mysql.com/downloads/connector/net/
- Stemmer.NET : https://stemmersnet.codeplex.com/SourceControl/list/changesets
- Log4Net : http://logging.apache.org/log4net/
- Newtonson JSON : https://www.nuget.org/packages/Newtonsoft.Json/
- ZedGraph : http://zedgraph.sourceforge.net/samples.html
